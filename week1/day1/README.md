# Day 1 - Getting Started with Groq API

## Overview

This project demonstrates how to make a basic chat completion request using the Groq API and Python.

## Concepts Learned

- Installing and using the Groq Python SDK
- Loading API keys securely using `python-dotenv`
- Creating a Groq client
- Sending a prompt to an LLM
- Reading the model's response

## Tech Stack

- Python
- Groq API
- python-dotenv

## Run

```bash
pip install groq python-dotenv
python hello_llm.py
```

## Learning Outcome

Built a simple LLM application capable of generating responses from a user prompt using the Groq API.
