# Day 2 - Prompt Engineering Basics

## Overview

This project explores the fundamentals of prompt engineering using the Groq API.

## Concepts Learned

- System Prompts
- User Prompts
- Role-based conversations
- Temperature parameter
- Generating creative responses

## Project

The program asks the model to act as a brand manager and generate a one-word name for a clothing company using carefully designed prompts.

## Tech Stack

- Python
- Groq API
- python-dotenv

## Run

```bash
pip install groq python-dotenv
python main.py
```

## Learning Outcome

Learned how system prompts influence model behavior and how temperature affects the creativity and randomness of the generated response.
