# Day 4 Assignment - AI Resume Parser

## 📌 Overview

Built a simple AI Resume Parser that extracts important information from a resume PDF using the Groq API and validates the output using Pydantic.

---

## 🚀 Features

- Read resume from a PDF
- Extract text using `pypdf`
- Generate structured JSON using the Groq API
- Validate the response with Pydantic

---

## 🛠️ Technologies

- Python
- Groq API
- Pydantic
- pypdf
- python-dotenv

---

## 📋 Extracted Information

- Name
- Email
- Skills
- Experience
- Internships
- Projects
- Achievements

---

## 📂 Workflow

```text
Resume PDF
    ↓
Extract Text
    ↓
Groq API
    ↓
JSON Output
    ↓
Pydantic Validation
```

---

## 🎯 What I Learned

- Reading PDF files in Python
- Extracting text from resumes
- Working with structured JSON outputs
- Using Pydantic for validation
- Building a simple AI-powered resume parser
